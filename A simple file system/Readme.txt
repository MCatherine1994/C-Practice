Understand the file system structure of MS-DOS and display the information of file system and the content of rout directory in the file system
Copy a file from the file system to the current directory in Linux, and copy a file from the current Linux directory into the root directory of the file system

Run:  
1. make
2. ./diskinfo disk.IMA
   ./disklist disk.IMA
   ./diskget disk.IMA (filename).(extension)
   ./diskput disk.IMA
